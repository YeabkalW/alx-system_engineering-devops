
# 0x13-firewall


This project is about configuring a firewall for your servers. You will be using the ufw firewall software.

## Objectives
Install the ufw firewall software.
Configure ufw to allow SSH access from your local computer.
Configure ufw to deny all other incoming traffic.
Configure ufw to allow HTTP and HTTPS traffic on port 80 and 443.
Test your firewall configuration.
